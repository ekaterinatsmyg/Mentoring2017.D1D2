Задание: 

Перед вами стоит задача написать кусочек редактора квестов для RPG игры. 
Редактор представляет собой набор готовых блоков с основными условиями и их результатами, которые может комбинировать между собой пользователь.</p> <p>Вы отвечаете за разработку бизнес-слоя, содержащего "блоки" квеста и их логику.
 Выберите подходящий паттерн(ы), которые вы будете использовать для реализации данной задачи.
 Обоснуйте свой выбор. Напишите простейший пример (без UI) для одного из блоков.
  Пример квеста в редакторе: 
   1.вызвать диалог 
   2. вывести текст ...
При выборе первого пункта добавить квест на выполение в таблицу квестов. 
Предложенное решение должно быть легко расширяемым для быстрого добавления новых элементов в редактор.


Возможное решение: 

Использовать паттерн: 

1. Наблюдатель. Для оповещения о завершении действий игрока после начала комманды и перехода к следующему шагу.Для оповещения Таблицы Квестов о начале Квеста.
2. Фабричный метод. Для создания новой команды основываясь на информации, полученной от игрока.
3. Комманда. Для управления действиями, сотсавляющими Квест.
